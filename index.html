Reinforcement Learning with Verifiable Rewards

nano-rlvr

Minimal implementation of RLVR — the technique behind DeepSeek-R1. Single GPU, no infra bloat, readable top-to-bottom. nanoGPT for RL alignment.

GitHub
The idea
RLHF
  • Train a reward model from human preferences
  • Reward model can be gamed or miscalibrated
  • Expensive data collection
  • Reward is a neural net forward pass
RLVR
  • Verify answers programmatically
  • Reward is exact: right or wrong
  • No preference data needed
  • Reward is a function call

RLVR works for tasks with checkable answers — math, logic, code. The model gets a problem, generates a solution, and the reward is 1 if correct, 0 if not. This has been shown to elicit chain-of-thought reasoning from base models.

Algorithms
REINFORCE
train_reinforce.py
  1. Generate a completion for each prompt
  2. Check the answer — reward = 0 or 1
  3. Subtract a running baseline to reduce variance
  4. Policy gradient update + KL penalty against reference
loss = -(reward - baseline) × log π(completion) + β · KL(π || πref)
GRPO
train_grpo.py — the DeepSeek-R1 algorithm
  1. Generate K completions per prompt (a “group”)
  2. Compute rewards for each completion
  3. Advantages = normalize rewards within the group
  4. PPO-style clipped surrogate optimization
  5. Multiple epochs per batch + KL penalty
advi = (ri - mean(rgroup)) / std(rgroup)
ratio = π(a|s) / πold(a|s)
loss = -min(ratio · adv, clip(ratio, 1−ε, 1+ε) · adv)

Verifiable reward

def check_arithmetic(completion, expected_answer):
    """Parse completion, extract final number, return 1.0 or 0.0."""
    extracted = _extract_number(completion)  # \boxed{}, after =, last number
    if extracted is None:
        return 0.0
    return 1.0 if abs(extracted - expected_answer) < 1e-6 else 0.0
Quickstart
$ uv sync
$ python train_reinforce.py  # simplest algorithm
$ python train_grpo.py  # DeepSeek-R1 algorithm

Reward climbs over ~100–200 steps. Sample completions are printed periodically — watch the model start reasoning step by step. Falls back to CPU if no CUDA.
